# Clara's Hub

일정과 할 일을 한 곳에서 관리하는 통합 플래너 PWA.

## 기능

- 월별 / 주별 캘린더 뷰
- 시간 있는 일정 + 시간 없는 할 일 분리 관리
- 카테고리별 색상 분류 (개인/부부/생일/학원/회사 기본)
- 할 일 마감일 + 임박/지남 시각 강조
- 데이터는 기기에 자동 저장 (localStorage)
- 오프라인 동작 지원 (Service Worker)
- 홈 화면에 추가하여 앱처럼 사용 가능

## 배포

GitHub Pages로 배포되어 있습니다.

## 모바일 설치 (PWA)

- **iOS Safari**: 공유 버튼 → "홈 화면에 추가"
- **Android Chrome**: 메뉴 → "홈 화면에 추가" 또는 "앱 설치"

## 기술

순수 HTML / CSS / JavaScript. 빌드 도구 없음.
